NaC Language

C ile yazılmış modern, küçük ve güçlü bir script dili

Hemen Başla

NaC Nedir?

NaC, C dili ile yazılmış, fonksiyonlar, döngüler, koşullar, diziler ve string işlemleri destekleyen hafif bir yorumlayıcı dildir. Öğrenmesi kolay, genişletmesi keyiflidir.

Özellikler

Fonksiyon tanımları (fn, rn)
If / Else ve For döngüleri
Dinamik diziler ve array literal
String + number otomatik işlemler
Yerleşik in() ve out()
Zaman fonksiyonu: time()

Örnek Kod

// Fibonacci
fn fib(n) {
  if (n < 2) {
    rn n;
  } : {
    rn fib(n-1) + fib(n-2);
  };
};

out(fib(10));

Başlarken

gcc -o nac nac.c -lm
./nac program.nac

Yukarıdaki komutlarla NaC yorumlayıcısını derleyip kendi .nac dosyalarını çalıştırabilirsin.
