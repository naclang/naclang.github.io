Getting Started

Install and compile NaC in minutes. Start building amazing things.

1 Prerequisites

Before installing NaC, make sure you have the following:

  • GCC compiler (version 4.9 or higher)
  • Make (optional, for build automation)
  • Git (to clone the repository)
üí° Quick Check: Run gcc --version to verify your GCC installation.

2 Download NaC

Clone the repository from GitHub:

git clone https://github.com/naclang/nac.git cd nac

Or download the latest release:

wget https://github.com/naclang/nac/releases/latest/download/nac.c wget https://github.com/naclang/nac/releases/latest/download/examples.zip

3 Compile the Interpreter

Compile the NaC interpreter using GCC:

gcc -o nac nac.c -lm
‚ö†Ô∏è Important: The -lm flag is required to link the math library.

Platform-Specific Instructions

Linux / macOS

# Compile gcc -o nac nac.c -lm # Make executable chmod +x nac # Optional: Install system-wide sudo cp nac /usr/local/bin/

Windows (MinGW / MSYS2)

# Using MinGW gcc -o nac.exe nac.c -lm # Add to PATH (optional) set PATH=%PATH%;C:\path\to\nac

4 Verify Installation

Create a simple test program to verify everything works:

// hello.nac out("Hello, NaC World!"); x = 42; out("The answer is: " + x);

Run your first NaC program:

./nac hello.nac
‚úÖ Success! If you see the output, NaC is working correctly.

5 Try More Examples

Explore the examples directory for more sample programs:

# Fibonacci sequence ./nac examples/fibonacci.nac # Array operations ./nac examples/arrays.nac # Sorting algorithm ./nac examples/bubble_sort.nac

Compilation Options

Advanced compilation options for optimization:

Debug Build

gcc -g -o nac-debug nac.c -lm

Optimized Build

gcc -O3 -o nac nac.c -lm

Static Build (Portable)

gcc -static -o nac nac.c -lm

Troubleshooting

Common Issues

Error: "undefined reference to 'sqrt'"
Solution: Make sure to include the -lm flag when compiling.
Error: "gcc: command not found"
Solution: Install GCC using your package manager:
